* Passo a Passo

- git init //cria um repositório
- git config user.name "seunome" // registro de usuário - nome do usuário
- git config user.email "umnome@umdominio" // registro de usuário - email
- git remote add origin "link repositório"
- git add . // adiciona todas as alterações ao repositório local (Stage)
- git push origin main // envia as alterações para o repositório remoto
